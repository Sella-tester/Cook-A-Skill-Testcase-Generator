# Cook-A-Skill-Testcase-Generator

## Project Overview
This repository contains the AI Skill: **Smart Test Case Generator**, developed for the "Cook A Skill" competition.

## Goal
To automatically transform Product Specifications (Product Spec) into comprehensive, intelligent, and structured Test Case lists.

## Features
- **Smart Analysis:** Parses logical flows and functional requirements.
- **Ambiguity Detection:** Identifies gaps in product specifications.
- **Automated Generation:** Creates Test Cases including steps, expected results, and priority.
- **Export Ready:** Outputs in Markdown format for easy integration into testing tools.

## Structure
- `SKILL.md`: The main AI instructions (Work in Progress).
- `spec.md`: The approved specification for this skill.
- `README.md`: This file.
